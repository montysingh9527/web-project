##### CSS常用属性
```
去掉ul和li前面的小黑点：list-style: none;
a标签去除下划线：text-decoration:none;
文字强制一行显示： white-space: nowrap;
vertical-align 调节图片垂直对齐的方式(去除图片与之间间隙)：vertical-align: middle;
禁止文本换行：white-space:nowrap;  		
禁止溢出：overflow:hidden;	 		
溢出部分显示省略号：text-overflow:ellipsis;	
input修改placeholder 的样式：  input::placeholder{}
```
